README e-LinkAcre

Overview

e-LinkAcre is a platform that connects Farmers, Government, and Buyers. first, it provide the information about the crops harvest, It promotes transparency, fair trade, and sustainable agriculture.

Pages

  • [Home] — Main landing page
  • [Farmer] — Farmer portal
  • [Government] — Government portal
  • [Buyer] — Buyer portal
  • [Updates] — Seasonal harvest info
  • [About Us] — Mission and values
  • [Login] — Sign up / account access

How to Use

Open index.html in a browser. Use the navigation bar to explore different sections. Farmers and buyers can check updates and manage their accounts, while government staff can access their portal and confirm information to be announced.
